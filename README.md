## 📝 About the Project
This project is a visual store management system. It allows users to simulate buying products, automatically updates inventory stock, and tracks transaction logs. The interface was designed using the NetBeans GUI Builder.

## ✨ Features
* **Transaction Window:** Select products from a dropdown, enter a quantity, and calculate totals.
* **Inventory Window:** View current stock levels, prices, and product names in a dynamically updated table.
* **Stock Management:** Prevents purchases if there isn't enough stock available.

## 🛠️ Technologies Used
* **Language:** Java
* **GUI Framework:** Java Swing
* **IDE:** Apache NetBeans

## 🚀 How to Run the App
1. Clone or download this repository to your computer.
2. Open the project folder in NetBeans.
3. In the Projects tab, find the `<default package>` or `Source Packages`.
4. Right-click on **`StoreAppDemo.java`** and select **Run File**.
